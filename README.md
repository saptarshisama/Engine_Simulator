# Engine Simulator

A web-based 3D engine simulator featuring multiple types of engines with interactive controls and real-time dyno graphs.

## Features

- **Interactive 3D Models**: Explore engines in 3D using your browser.
- **Multiple Engine Types**:
  - Cross-Plane V8
  - Inline-6
  - V12 (60°)
  - Radial-9 Aviation
  - Bugatti W16
  - Stirling Engine
  - Steam Engine
- **Configurable Settings**: Adjust RPM, view wireframes, and control other engine-specific parameters.
- **Real-Time Dyno**: Visualizes RPM, Torque, and Power for the running engine.

## Technologies Used

- **HTML/CSS/JavaScript**: Vanilla web technologies for the frontend.
- **Three.js**: Used for rendering the 3D engine models and animations.
- **lil-gui**: For adjusting engine parameters dynamically.

## Usage

Simply open `index.html` in your modern web browser to start exploring the engines. No local server required!
